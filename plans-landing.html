왜 무료를 3회까지만 주나요?
API 키를 사용자 기기에 두지 않고 서버에서만 호출하기 때문에, 무제한 무료는 비용 통제가 어렵습니다. 3회 체험 후 Pro로 넘기는 구조가 가장 안전합니다.
왜 월 300회인가요?
초기 사용량과 API 원가를 고려했을 때, 개인 유저가 충분히 체감하면서도 운영자가 적자 위험을 낮출 수 있는 가장 현실적인 상한입니다.
토스페이먼츠는 언제 붙이나요?
현재는 랜딩과 정책이 먼저 준비된 상태입니다. 다음 단계는 토스페이먼츠 결제창, webhook, Pro 코드 또는 구독 반영을 연결하는 것입니다.
이 랜딩은 Chrome Web Store 외부 결제 유도를 위한 정적 페이지입니다. 실제 판매 전에는 토스페이먼츠 버튼, 개인정보처리방침, 환불 정책, 고객 문의 메일을 함께 연결하는 것이 맞습니다.
