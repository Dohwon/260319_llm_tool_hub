# Current Initiatives

## 모델/툴 추천 운영 체계

- 상태: 진행 중
- 목표: 프로젝트 설명만으로 모델/툴/설치 경로/토큰 리밋/메모리 공유 전략을 함께 제안하는 허브와 전역 스킬 구축
- 기대 효과:
  - 새 프로젝트 착수 시 도구 선택 시간을 줄임
  - Codex/Claude Code/Gemini/Cursor/Windsurf/Ollama 등 선택 기준을 표준화
  - 사용자 환경별 설치 경로를 초보자도 따라갈 수 있는 형태로 제공
  - 전역 메모리를 특정 벤더 기능이 아니라 파일 어댑터 중심으로 통합
  - 모델 간 handover와 shared memory 상태를 시각적으로 추적 가능하게 만듦
  - planner-agent-ko / idea-agent-ko 산출물을 전역 메모리와 협업 패키지에 같이 올려 다른 모델/사람이 이어받을 수 있게 함
